This website is about Blacksmithing and how we can further the possibilities and help blacksmithing for the better.

Blacksmiths were first called hittites, blacksmiths had used copper for a long time but after some years they started working with other metals to see which is the strongest and that is when they started using iron. Blacksmiths had started using iron around 1500 Bc and the tools they had used did not changed much since when they started. But in some time they had used other tools when blacksmithing, like using the chisel or a flatter which squishes the metal and makes it flat and more able to cut or bend it to other shapes such as a shovel or kitchen knife
